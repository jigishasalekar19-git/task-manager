Cozy Task Manager

A beautifully designed, highly interactive, and responsive vanilla JavaScript Task Manager application. Built with a charming pastel "glassmorphism" UI, this app offers robust task management features paired with state-driven architecture and locally persistent data.
Features
- Full CRUD Operations: Seamlessly add, edit, view, and delete tasks.
- State-Driven Completion: Easily toggle tasks between active and completed statuses.
- **Due Dates**: Assign specific deadlines to organize your daily schedule.
- Advanced State Filtering: Instantly filter list views by 'All', 'Active', or 'Done' metrics.
- Real-Time Task Counter: Dynamically monitors and displays remaining open tasks.
- Cozy & Midnight Themes: A fully automated dark-mode toggle featuring soft-on-the-eyes pastel color themes.
- Local Data Persistence: Built-in synchronization with `localStorage` so tasks remain saved even after closing the browser.
- XSS Security Protection: Strict sanitization of client strings to prevent arbitrary script injections.

Tech Stack Used
- **HTML5**: Semantic document structuring.
- **CSS3**: Custom modern Flexbox grid layouts, variable-driven color theming, glassmorphic filters, and interactive transition states.
- **Vanilla JavaScript (ES6+)**: Structural state rendering, data management algorithms, browser API interfacing, and client-side logic.
- **Google Fonts**: Custom high-fidelity 'Fredoka' typography integration.

 How to Run the Project

You can run this project locally on your machine in just two steps:

Method 1: The Quick Start
1. Clone or download this repository to your local directory.
2. Locate the `index.html` file and double-click it.


Method 2: Live Server Development (for VSCODE)
1. Launch the project directory inside **Visual Studio Code**.
2. Install the **Live Server** extension by *Ritwick Dey* from the extensions marketplace.
3. Open `index.html`, then click the **"Go Live"** button located at the bottom-right status bar of your editor window.
